# 编辑多维表

## 功能描述

编辑多维表是系统的核心功能之一，允许用户对已创建的多维表进行修改，包括添加列、编辑列、添加行、在指定位置插入多行以及编辑行数据等操作。这些功能需要在Flutter移动端、Electron桌面端和Rust命令行工具中实现。

## 界面流程

### 1. 移动端（Flutter）

#### 1.1 添加列
1. **入口**：在表编辑界面点击“添加列”按钮
2. **列信息输入**：
   - 输入列名称
   - 选择列数据类型（文本、数字、单选、多选、超级链接、日期/时间、布尔值）
   - 设置默认值（可选）
   - 设置是否必填（可选）
   - 对于单选和多选类型，设置选项列表
3. **确认添加**：点击“添加”按钮，完成列的添加

#### 1.2 编辑列
1. **入口**：在表编辑界面长按列标题，选择“编辑列”选项
2. **编辑列信息**：
   - 修改列名称
   - 修改列数据类型
   - 修改默认值
   - 修改是否必填
   - 修改选项列表（对于单选和多选类型）
3. **确认修改**：点击“保存”按钮，完成列的编辑

#### 1.3 添加一行
1. **入口**：在表编辑界面点击“添加行”按钮
2. **输入行数据**：为每个列输入相应的数据
3. **确认添加**：点击“保存”按钮，完成行的添加

#### 1.4 上方/下方插入多行
1. **入口**：在表编辑界面长按某行，选择“插入行”选项
2. **设置插入参数**：
   - 选择插入位置（上方/下方）
   - 输入插入行数
3. **确认插入**：点击“插入”按钮，完成多行的插入

#### 1.5 编辑行数据
1. **入口**：在表编辑界面点击要编辑的单元格
2. **修改数据**：输入新的数据值
3. **确认修改**：点击“保存”按钮或点击其他单元格，完成数据的编辑

// src/db/field.rs
use rusqlite::{Transaction, Result};
use serde_json::to_string;
use crate::types::FieldInfo;

pub fn update_field_info(tx: &Transaction, field_info: &FieldInfo) -> Result<String> {
    let sql = "UPDATE fields SET field_name = ?, field_type = ?, properties = ?, updated_at = ? WHERE field_id = ?";
    
    let properties_json = to_string(&field_info.properties)?;
    
    tx.execute(
        sql,
        &[&field_info.field_name, &field_info.field_type, &properties_json, 
          &field_info.updated_at.to_rfc3339(), &field_info.field_id],
    )?;
    
    Ok(field_info.field_id.clone())
}

// src/db/row.rs
use rusqlite::{Transaction, Result};
use crate::types::{RowData, InsertRowsRequest};
use chrono::{DateTime, Utc};

pub fn insert_row(tx: &Transaction, table_id: &str, row_data: &RowData) -> Result<String> {
    let table_name = format!("data_{}", table_id);
    
    // 构建插入语句
    let mut columns = vec!["id", "created_at", "updated_at"];
    let mut values = vec![&row_data.id, &row_data.created_at.to_rfc3339(), &row_data.updated_at.to_rfc3339()];
    let mut placeholders = vec!["?", "?", "?"];
    
    for (field_id, value) in &row_data.data {
        columns.push(field_id);
        values.push(value);
        placeholders.push("?");
    }
    
    let columns_str = columns.join(", ");
    let placeholders_str = placeholders.join(", ");
    
    let sql = format!(
        "INSERT INTO {} ({}) VALUES ({})",
        table_name, columns_str, placeholders_str
    );
    
    tx.execute(&sql, values.as_slice())?;
    
    Ok(row_data.id.clone())
}

pub fn update_row(tx: &Transaction, table_id: &str, row_data: &RowData) -> Result<String> {
    let table_name = format!("data_{}", table_id);
    
    // 构建更新语句
    let mut set_clauses = vec!["updated_at = ?"];
    let mut values = vec![&row_data.updated_at.to_rfc3339()];
    
    for (field_id, value) in &row_data.data {
        set_clauses.push(format!("{} = ?", field_id));
        values.push(value);
    }
    
    values.push(&row_data.id);
    
    let set_clauses_str = set_clauses.join(", ");
    
    let sql = format!(
        "UPDATE {} SET {} WHERE id = ?",
        table_name, set_clauses_str
    );
    
    tx.execute(&sql, values.as_slice())?;
    
    Ok(row_data.id.clone())
}

pub fn insert_multiple_rows(tx: &Transaction, request: &InsertRowsRequest) -> Result<Vec<RowData>> {
    let table_name = format!("data_{}", request.table_id);
    let mut rows = Vec::new();
    
    // 创建指定数量的空行
    for _ in 0..request.row_count {
        let row = RowData::new();
        rows.push(row);
    }
    
    // 插入所有行
    for row in &rows {
        insert_row(tx, &request.table_id, row)?;
    }
    
    Ok(rows)
}

class _EditTableScreenState extends State<EditTableScreen> {
  List<FieldInfo> _fields = [];
  List<RowData> _rows = [];
  bool _isLoading = true;
  
  @override
  void initState() {
    super.initState();
    _loadTableData();
  }
  
  Future<void> _loadTableData() async {
    // 加载字段信息
    _fields = await FieldService.getFields(widget.tableId);
    // 加载行数据
    _rows = await RowService.getRows(widget.tableId);
    setState(() {
      _isLoading = false;
    });
  }
  
  Future<void> _addColumn() async {
    // 导航到添加列页面
    var result = await Navigator.push(
      context,
      MaterialPageRoute(builder: (context) => AddColumnScreen(tableId: widget.tableId)),
    );
    
    if (result != null) {
      // 重新加载字段信息
      _fields = await FieldService.getFields(widget.tableId);
      setState(() {});
    }
  }
  
  Future<void> _editColumn(FieldInfo field) async {
    // 导航到编辑列页面
    var result = await Navigator.push(
      context,
      MaterialPageRoute(builder: (context) => EditColumnScreen(field: field)),
    );
    
    if (result != null) {
      // 重新加载字段信息
      _fields = await FieldService.getFields(widget.tableId);
      setState(() {});
    }
  }
  
  Future<void> _addRow() async {
    // 导航到添加行页面
    var result = await Navigator.push(
      context,
      MaterialPageRoute(builder: (context) => AddRowScreen(tableId: widget.tableId, fields: _fields)),
    );
    
    if (result != null) {
      // 重新加载行数据
      _rows = await RowService.getRows(widget.tableId);
      setState(() {});
    }
  }
  
  Future<void> _insertRows(int index, String position) async {
    // 显示插入行对话框
    showDialog(
      context: context,
      builder: (context) {
        int rowCount = 1;
        return AlertDialog(
          title: Text('插入行'),
          content: Column(
            mainAxisSize: MainAxisSize.min,
            children: [
              Text('请输入要插入的行数:'),
              TextField(
                keyboardType: TextInputType.number,
                onChanged: (value) {
                  rowCount = int.tryParse(value) ?? 1;
                },
                decoration: InputDecoration(hintText: '行数'),
              ),
            ],
          ),
          actions: [
            TextButton(
              onPressed: () => Navigator.pop(context),
              child: Text('取消'),
            ),
            TextButton(
              onPressed: () async {
                Navigator.pop(context);
                // 调用API插入行
                var request = InsertRowsRequest(
                  tableId: widget.tableId,
                  position: position,
                  referenceRowId: _rows[index].id,
                  rowCount: rowCount,
                );
                await RowService.insertRows(request);
                // 重新加载行数据
                _rows = await RowService.getRows(widget.tableId);
                setState(() {});
              },
              child: Text('插入'),
            ),
          ],
        );
      },
    );
  }
  
  Future<void> _editRow(RowData row) async {
    // 导航到编辑行页面
    var result = await Navigator.push(
      context,
      MaterialPageRoute(builder: (context) => EditRowScreen(tableId: widget.tableId, row: row, fields: _fields)),
    );
    
    if (result != null) {
      // 重新加载行数据
      _rows = await RowService.getRows(widget.tableId);
      setState(() {});
    }
  }
  
  @override
  Widget build(BuildContext context) {
    if (_isLoading) {
      return Center(child: CircularProgressIndicator());
    }
    
    return Scaffold(
      appBar: AppBar(
        title: Text('编辑多维表'),
        actions: [
          IconButton(
            icon: Icon(Icons.add),
            onPressed: _addColumn,
            tooltip: '添加列',

            scrollDirection: Axis.horizontal,
            child: DataTable(
              columns: [
                DataColumn(label: Text('操作')),
                ..._fields.map((field) => DataColumn(
                  label: GestureDetector(
                    onLongPress: () => _editColumn(field),
                    child: Text(field.fieldName),
                  ),
                )).toList(),
              ],
              rows: _rows.asMap().entries.map((entry) {
                int index = entry.key;
                RowData row = entry.value;
                return DataRow(
                  cells: [
                    DataCell(
                      PopupMenuButton(
                        itemBuilder: (context) => [
                          PopupMenuItem(
                            child: Text('编辑'),
                            value: 'edit',
                          ),
                          PopupMenuItem(
                            child: Text('在上方插入行'),
                            value: 'insert_above',
                          ),
                          PopupMenuItem(
                            child: Text('在下方插入行'),
                            value: 'insert_below',
                          ),
                        ],
                        onSelected: (value) {
                          switch (value) {
                            case 'edit':
                              _editRow(row);
                              break;
                            case 'insert_above':
                              _insertRows(index, 'above');
                              break;
                            case 'insert_below':
                              _insertRows(index, 'below');
                              break;
                          }
                        },
                      ),
                    ),
                    ..._fields.map((field) => DataCell(
                      Text(row.data[field.fieldId] ?? ''),
                    )).toList(),
                  ],
                );
              }).toList(),
            ),
          ),
        ],
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: _addRow,
        child: Icon(Icons.add),
        tooltip: '添加行',
      ),
    );
  }
}

## 验证规则

### 1. 列信息验证
- **列名称**：必填，长度不超过30个字符
- **列类型**：必填，必须是系统支持的类型
- **默认值**：根据列类型进行验证
  - 文本：任意字符串
  - 数字：必须是有效的数字
  - 单选：必须是选项列表中的值
  - 多选：必须是选项列表中的值，多个值用逗号分隔
  - 超级链接：必须是有效的URL
  - 日期/时间：必须是有效的日期时间格式
  - 布尔值：必须是true或false

### 2. 行数据验证
