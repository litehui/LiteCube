# 临时文件管理

## 功能描述

临时文件管理是系统的重要功能之一，确保在编辑多维表时：
- 创建新的多维表时，先在系统临时文件夹（TEMP）中创建临时.cl文件
- 编辑已有的.cl文件时，复制到临时文件夹中作为工作副本
- 只有在保存或另存为时，才将数据写入用户指定的文件位置
- 确保原始文件的安全性

## 界面流程

### 1. 移动端（Flutter）

#### 1.1 创建新文件流程
1. 用户点击"新建表"按钮
2. 系统在TEMP文件夹中创建唯一名称的临时.cl文件
3. 用户输入表信息和字段
4. 用户点击"保存"或"另存为"时，弹出文件选择器
5. 用户选择保存位置和文件名
6. 临时文件复制到用户指定位置
7. 关闭临时文件，切换到正式文件

#### 1.2 打开已有文件流程
1. 用户点击"打开表"按钮
2. 弹出文件选择器，用户选择.cl文件
3. 系统将文件复制到TEMP文件夹中，添加临时文件名
4. 在应用中打开临时文件进行编辑
5. 用户点击"保存"时，更新临时文件
6. 用户点击"另存为"时，弹出文件选择器
7. 用户点击"退出"或应用被清理时，删除临时文件

```rust
use chrono::{DateTime, Utc};
use std::path::{Path, PathBuf};

#[derive(Debug, Serialize, Deserialize, Clone)]
pub struct FileState {
    pub file_path: Option<PathBuf>,
    pub temp_file_path: PathBuf,
    pub is_unsaved: bool,
    pub original_exists: bool,
    pub created_at: DateTime<Utc>,
    pub last_modified: DateTime<Utc>,
}

impl FileState {
    pub fn new(temp_path: PathBuf) -> Self {
        let now = Utc::now();
        Self {
            file_path: None,
            temp_file_path: temp_path,
            is_unsaved: true,
            original_exists: false,
            created_at: now,
            last_modified: now,
        }
    }

    pub fn from_original(original_path: PathBuf, temp_path: PathBuf) -> Self {
        let now = Utc::now();
        Self {
            file_path: Some(original_path),
            temp_file_path: temp_path,
            is_unsaved: false,
            original_exists: true,
            created_at: now,
            last_modified: now,
        }
    }

    pub fn get_db_path(&self) -> &PathBuf {
        &self.temp_file_path
    }
}
```

#### 1.2 临时文件管理接口

```rust
// src/api/file.rs
use std::fs::{self, File};
use std::io::{self, Read, Write};
use std::path::{Path, PathBuf};
use uuid::Uuid;
use std::env;
use crate::types::file::FileState;
use chrono::Utc;

/// 获取系统临时文件夹路径
fn get_temp_dir() -> PathBuf {
    let mut temp_dir = env::temp_dir();
    temp_dir.push("cl-table");
    // 确保临时文件夹存在
    if !temp_dir.exists() {
        fs::create_dir_all(&temp_dir).unwrap();
    }
    temp_dir
}

/// 创建唯一的临时文件名
fn create_temp_filename() -> String {
    let uuid = Uuid::new_v4().to_string();
    format!("cl-table-{}.cl", uuid)
}

/// 创建新的临时文件，用于新建表
pub fn create_new_temp_file() -> Result<FileState, io::Error> {
    let temp_dir = get_temp_dir();
    let temp_filename = create_temp_filename();
    let temp_file_path = temp_dir.join(temp_filename);
    
    // 创建空的临时文件（后面会通过db操作创建表结构）
    let file = File::create(&temp_file_path)?;
    drop(file);
    
    let file_state = FileState::new(temp_file_path);
    Ok(file_state)
}

/// 从已有文件创建临时文件，用于打开表
pub fn create_temp_from_original(original_path: &Path) -> Result<FileState, io::Error> {
    let temp_dir = get_temp_dir();
    let temp_filename = create_temp_filename();
    let temp_file_path = temp_dir.join(temp_filename);
    
    // 复制原始文件到临时位置
    fs::copy(original_path, &temp_file_path)?;
    
    let file_state = FileState::from_original(original_path.to_path_buf(), temp_file_path);
    Ok(file_state)
}

/// 保存临时文件到指定路径
pub fn save_file(file_state: &FileState, target_path: &Path) -> Result<(), io::Error> {
    fs::copy(&file_state.temp_file_path, target_path)?;
    
    let mut new_file_state = file_state.clone();
    new_file_state.file_path = Some(target_path.to_path_buf());
    new_file_state.is_unsaved = false;
    new_file_state.last_modified = Utc::now();
    
    Ok(())
}

/// 另存为到新位置
pub fn save_as(file_state: &mut FileState, new_path: &Path) -> Result<(), io::Error> {
    fs::copy(&file_state.temp_file_path, new_path)?;
    
    file_state.file_path = Some(new_path.to_path_buf());
    file_state.is_unsaved = false;
    file_state.last_modified = Utc::now();
    
    Ok(())
}

/// 删除临时文件
pub fn cleanup_temp_file(file_state: &FileState) -> Result<(), io::Error> {
    if file_state.temp_file_path.exists() {
        fs::remove_file(&file_state.temp_file_path)?;
    }
    Ok(())
}

/// 清理所有过期的临时文件（启动时调用
pub fn cleanup_old_temp_files() -> Result<(), io::Error> {
    let temp_dir = get_temp_dir();
    if !temp_dir.exists() {
        return Ok(());
    }
    
    let twenty_four_hours_ago = Utc::now() - chrono::Duration::hours(24);
    
    for entry in fs::read_dir(temp_dir)? {
        let entry = entry?;
        let path = entry.path();
        if let Some(filename) = path.file_name() {
            let filename_str = filename.to_string_lossy();
            if filename_str.starts_with("cl-table-") && filename_str.ends_with(".cl") {
                if let Ok(metadata) = fs::metadata(&path) {
                    if let Ok(modified) = metadata.modified() {
                        let modified_datetime: DateTime<Utc> = modified.into();
                        if modified_datetime < twenty_four_hours_ago {
                            let _ = fs::remove_file(&path);
                        }
                    }
                }
            }
        }
    }
    
    Ok(())
}
```

```dart
// lib/services/file_service.dart
import 'dart:io';
import 'package:path/path.dart' as path;
import 'package:path_provider/path_provider.dart';
import 'package:file_picker/file_picker.dart';

class FileService {
  static Future<FileState> createNewTempFile() async {
    final tempDir = await getTemporaryDirectory();
    final tempDirCl = Directory(path.join(tempDir.path, 'cl-table'));
    if (!await tempDirCl.exists()) {
      await tempDirCl.create(recursive: true);
    }
    final tempFileName = 'cl-table-${DateTime.now().millisecondsSinceEpoch}.cl';
    final tempFile = File(path.join(tempDirCl.path, tempFileName));
    await tempFile.create();
    
    return FileState(
      tempFilePath: tempFile.path,
      isUnsaved: true,
      originalExists: false,
    );
  }

  static Future<FileState> createTempFromOriginal(String originalPath) async {
    final tempDir = await getTemporaryDirectory();
    final tempDirCl = Directory(path.join(tempDir.path, 'cl-table'));
    if (!await tempDirCl.exists()) {
      await tempDirCl.create(recursive: true);
    }
    final tempFileName = 'cl-table-${DateTime.now().millisecondsSinceEpoch}.cl';
    final tempFile = File(path.join(tempDirCl.path, tempFileName));
    
    final originalFile = File(originalPath);
    await originalFile.copy(tempFile.path);
    
    return FileState(
      filePath: originalPath,
      tempFilePath: tempFile.path,
      isUnsaved: false,
      originalExists: true,
    );
  }

  static Future<void> saveFile(FileState fileState, String targetPath) async {
    final tempFile = File(fileState.tempFilePath);
    await tempFile.copy(targetPath);
  }

  static Future<String?> pickSaveLocation() async {
    final result = await FilePicker.platform.saveFile(
      dialogTitle: '保存多维表',
      fileName: 'untitled.cl',
      type: FileType.custom,
      allowedExtensions: ['cl'],
    );
    return result;
  }

  static Future<String?> pickOpenFile() async {
    final result = await FilePicker.platform.pickFiles(
      type: FileType.custom,
      allowedExtensions: ['cl'],
    );
    if (result != null && result.files.single.path != null) {
      return result.files.single.path!;
    }
    return null;
  }

  static Future<void> cleanupTempFile(FileState fileState) async {
    final tempFile = File(fileState.tempFilePath);
    if (await tempFile.exists()) {
      await tempFile.delete();
    }
  }
}

class FileState {
  final String? filePath;
  final String tempFilePath;
  final bool isUnsaved;
  final bool originalExists;

  FileState({
    this.filePath,
    required this.tempFilePath,
    required this.isUnsaved,
    required this.originalExists,
  });

  FileState copyWith({
    String? filePath,
    String? tempFilePath,
    bool? isUnsaved,
    bool? originalExists,
  }) {
    return FileState(
      filePath: filePath ?? this.filePath,
      tempFilePath: tempFilePath ?? this.tempFilePath,
      isUnsaved: isUnsaved ?? this.isUnsaved,
      originalExists: originalExists ?? this.originalExists,
    );
  }
}
```

#### 2.2 Electron桌面端

```javascript
// src/main/services/file_service.js
const fs = require('fs');
const path = require('path');
const { dialog } = require('electron');
const os = require('os');

class FileService {
  static getTempDir() {
    const tempDir = path.join(os.tmpdir(), 'cl-table');
    if (!fs.existsSync(tempDir)) {
      fs.mkdirSync(tempDir, { recursive: true });
    }
    return tempDir;
  }

  static createNewTempFile() {
    const tempDir = this.getTempDir();
    const tempFileName = `cl-table-${Date.now()}.cl`;
    const tempFilePath = path.join(tempDir, tempFileName);
    
    fs.closeSync(fs.openSync(tempFilePath, 'w'));
    
    return {
      filePath: null,
      tempFilePath,
      isUnsaved: true,
      originalExists: false,
    };
  }

  static createTempFromOriginal(originalPath) {
    const tempDir = this.getTempDir();
    const tempFileName = `cl-table-${Date.now()}.cl`;
    const tempFilePath = path.join(tempDir, tempFileName);
    
    fs.copyFileSync(originalPath, tempFilePath);
    
    return {
      filePath: originalPath,
      tempFilePath,
      isUnsaved: false,
      originalExists: true,
    };
  }

  static saveFile(fileState, targetPath) {
    fs.copyFileSync(fileState.tempFilePath, targetPath);
  }

  static async pickSaveLocation() {
    const result = await dialog.showSaveDialog({
      title: '保存多维表',
      defaultPath: 'untitled.cl',
      filters: [{ name: '多维表文件', extensions: ['cl'] }],
    });
    
    if (!result.canceled && result.filePath) {
      return result.filePath;
    }
    return null;
  }

  static async pickOpenFile() {
    const result = await dialog.showOpenDialog({
      title: '打开多维表',
      filters: [{ name: '多维表文件', extensions: ['cl'] }],
    });
    
    if (!result.canceled && result.filePaths.length > 0) {
      return result.filePaths[0];
    }
    return null;
  }

  static cleanupTempFile(fileState) {
    if (fs.existsSync(fileState.tempFilePath)) {
      fs.unlinkSync(fileState.tempFilePath);
    }
  }

  static cleanupOldTempFiles() {
    const tempDir = this.getTempDir();
    const now = Date.now();
    const twentyFourHoursAgo = now - (24 * 60 * 60 * 1000);
    
    fs.readdirSync(tempDir).forEach(filename => {
      if (filename.startsWith('cl-table-') && filename.endsWith('.cl')) {
        const filePath = path.join(tempDir, filename);
        const stat = fs.statSync(filePath);
        if (stat.mtimeMs < twentyFourHoursAgo) {
          fs.unlinkSync(filePath);
        }
      }
    });
  }
}

module.exports = FileService;
```

## 验证规则

### 1. 文件操作验证

- 临时文件命名规则确保唯一性，避免冲突
- 保存时检查目标路径是否有效且可写
- 另存为新文件时，检查文件名是否符合要求（.cl后缀）
- 打开文件时，验证是否为有效的.cl文件

## 错误处理

### 1. 前端错误处理

- 临时文件创建失败，显示错误提示
- 文件保存失败，显示错误并提示用户重试
- 文件打开失败，显示错误信息
- 临时文件清理失败，记录日志但不影响用户使用

### 2. 后端错误处理

- 文件操作错误，返回标准化的错误信息
- 临时文件操作失败，回滚到安全状态

## 性能优化

### 1. 临时文件管理优化

- 启动时清理超过24小时的临时文件，避免占用空间
- 使用唯一的临时文件名，确保不会覆盖重要文件
- 临时文件操作使用内存缓存，减少磁盘I/O
- 批量写入操作优化，减少不必要的文件复制
