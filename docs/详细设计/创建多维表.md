# 创建多维表

## 功能描述

创建多维表是系统的核心功能之一，允许用户创建新的多维表结构，定义字段类型和属性，为后续的数据管理做准备。

## 界面流程

### 1. 移动端（Flutter）

1. **入口**：从主界面点击“创建表”按钮
2. **表信息输入**：
   - 输入表名称
   - 选择表类型（默认多维表）
   - 输入表描述（可选）
3. **字段添加**：
   - 点击“添加字段”按钮
   - 选择字段类型（文本、数字、单选、多选、超级链接等）
   - 输入字段名称
   - 设置字段属性（如默认值、是否必填等）
   - 重复添加多个字段
4. **确认创建**：点击“创建”按钮，完成表的创建

### 2. 桌面端（Electron）

1. **入口**：从主界面点击“创建表”按钮
2. **表信息输入**：
   - 输入表名称
   - 选择表类型（默认多维表）
   - 输入表描述（可选）
3. **字段添加**：
   - 点击“添加字段”按钮
   - 选择字段类型（文本、数字、单选、多选、超级链接等）
   - 输入字段名称
   - 设置字段属性（如默认值、是否必填等）
   - 重复添加多个字段
4. **确认创建**：点击“创建”按钮，完成表的创建

#### 1.3 数据库操作

```rust
// src/db/table.rs
pub fn insert_table_info(tx: &Transaction, table_info: &TableInfo) -> Result<String, Error> {
    let sql = "INSERT INTO tables (table_id, table_name, table_type, description, created_at, updated_at) 
               VALUES (?, ?, ?, ?, ?, ?)";
    
    tx.execute(
        sql,
        &[&table_info.table_id, &table_info.table_name, &table_info.table_type, 
          &table_info.description, &table_info.created_at, &table_info.updated_at],
    )?;
    
    Ok(table_info.table_id.clone())
}

// src/db/field.rs
pub fn insert_field_info(tx: &Transaction, field_info: &FieldInfo) -> Result<String, Error> {
    let sql = "INSERT INTO fields (field_id, table_id, field_name, field_type, properties, created_at, updated_at) 
               VALUES (?, ?, ?, ?, ?, ?, ?)";
    
    let properties_json = serde_json::to_string(&field_info.properties)?;
    
    tx.execute(
        sql,
        &[&field_info.field_id, &field_info.table_id, &field_info.field_name, 
          &field_info.field_type, &properties_json, &field_info.created_at, &field_info.updated_at],
    )?;
    
    Ok(field_info.field_id.clone())
}
```

### 2. 前端实现

#### 2.1 Flutter移动端

```dart
// lib/views/create_table.dart
class CreateTableScreen extends StatefulWidget {
  @override
  _CreateTableScreenState createState() => _CreateTableScreenState();
}

class _CreateTableScreenState extends State<CreateTableScreen> {
  final _formKey = GlobalKey<FormState>();
  String _tableName = '';
  String _description = '';
  List<Field> _fields = [];
  
  void _addField() {
    setState(() {
      _fields.add(Field());
    });
  }
  
  void _createTable() async {
    if (_formKey.currentState!.validate()) {
      _formKey.currentState!.save();
      
      // 构建表信息
      var tableInfo = TableInfo(
        tableId: Uuid().v4(),
        tableName: _tableName,
        tableType: '多维表',
        description: _description,
        createdAt: DateTime.now().toUtc(),
        updatedAt: DateTime.now().toUtc(),
      );
      
      // 调用API创建表
      var result = await TableService.createTable(tableInfo);
      
      if (result.success) {
        // 为每个字段调用API添加字段
        for (var field in _fields) {
          var fieldInfo = FieldInfo(
            fieldId: Uuid().v4(),
            tableId: tableInfo.tableId,
            fieldName: field.name,
            fieldType: field.type,
            properties: field.properties,
            createdAt: DateTime.now().toUtc(),
            updatedAt: DateTime.now().toUtc(),
          );
          await FieldService.addField(fieldInfo);
        }
        
        // 导航回主界面
        Navigator.pop(context, true);
      } else {
        // 显示错误信息
        ScaffoldMessenger.of(context).showSnackBar(
          SnackBar(content: Text('创建表失败: ${result.error}')),
        );
      }
    }
  }
  
  @override
  Widget build(BuildContext context) {
    // 构建界面
    return Scaffold(
      appBar: AppBar(title: Text('创建多维表')),
      body: Form(
        key: _formKey,
        child: ListView(
          padding: EdgeInsets.all(16),
          children: [
            TextFormField(
              decoration: InputDecoration(labelText: '表名称'),
              validator: (value) => value!.isEmpty ? '请输入表名称' : null,
              onSaved: (value) => _tableName = value!,
            ),
            TextFormField(
              decoration: InputDecoration(labelText: '表描述'),
              onSaved: (value) => _description = value!,
            ),
            SizedBox(height: 20),
            Text('字段', style: TextStyle(fontSize: 18, fontWeight: FontWeight.bold)),
            ..._fields.asMap().entries.map((entry) {
              int index = entry.key;
              Field field = entry.value;
              return FieldEditor(
                field: field,
                onUpdate: (updatedField) {
                  setState(() {
                    _fields[index] = updatedField;
                  });
                },
                onDelete: () {
                  setState(() {
                    _fields.removeAt(index);
                  });
                },
              );
            }).toList(),
            ElevatedButton(
              onPressed: _addField,
              child: Text('添加字段'),
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: _createTable,
              child: Text('创建表'),
            ),
          ],
        ),
      ),
    );
  }
}
```

#### 2.2 Electron桌面端

```javascript
// src/renderer/components/CreateTable.js
import React, { useState } from 'react';
import { TableService, FieldService } from '../services';
import { v4 as uuidv4 } from 'uuid';

const CreateTable = ({ onClose }) => {
  const [tableName, setTableName] = useState('');
  const [description, setDescription] = useState('');
  const [fields, setFields] = useState([]);
  const [error, setError] = useState('');

  const addField = () => {
    setFields([...fields, { id: uuidv4(), name: '', type: '文本', properties: {} }]);
  };

  const updateField = (index, updatedField) => {
    const newFields = [...fields];
    newFields[index] = updatedField;
    setFields(newFields);
  };

  const deleteField = (index) => {
    const newFields = [...fields];
    newFields.splice(index, 1);
    setFields(newFields);
  };

  const createTable = async () => {
    if (!tableName) {
      setError('请输入表名称');
      return;
    }

    try {
      // 构建表信息
      const tableInfo = {
        table_id: uuidv4(),
        table_name: tableName,
        table_type: '多维表',
        description: description,
        created_at: new Date().toISOString(),
        updated_at: new Date().toISOString(),
      };

      // 调用API创建表
      const result = await TableService.createTable(tableInfo);

      if (result.success) {
        // 为每个字段调用API添加字段
        for (const field of fields) {
          const fieldInfo = {
            field_id: field.id,
            table_id: tableInfo.table_id,
            field_name: field.name,
            field_type: field.type,
            properties: field.properties,
            created_at: new Date().toISOString(),
            updated_at: new Date().toISOString(),
          };
          await FieldService.addField(fieldInfo);
        }

        // 关闭对话框
        onClose(true);
      } else {
        setError(`创建表失败: ${result.error}`);
      }
    } catch (err) {
      setError(`创建表失败: ${err.message}`);
    }
  };

  return (
    <div className="create-table">
      <h2>创建多维表</h2>
      {error && <div className="error">{error}</div>}
      <form>
        <div className="form-group">
          <label>表名称</label>
          <input
            type="text"
            value={tableName}
            onChange={(e) => setTableName(e.target.value)}
          />
        </div>
        <div className="form-group">
          <label>表描述</label>
          <textarea
            value={description}
            onChange={(e) => setDescription(e.target.value)}
          />
        </div>
        <div className="form-group">
          <label>字段</label>
          {fields.map((field, index) => (
            <FieldEditor
              key={field.id}
              field={field}
              onUpdate={(updatedField) => updateField(index, updatedField)}
              onDelete={() => deleteField(index)}
            />
          ))}
          <button type="button" onClick={addField} className="add-field-btn">
            添加字段
          </button>
        </div>
        <div className="form-actions">
          <button type="button" onClick={() => onClose(false)} className="cancel-btn">
            取消
          </button>
          <button type="button" onClick={createTable} className="create-btn">
            创建
          </button>
        </div>
      </form>
    </div>
  );
};

export default CreateTable;
```

## 验证规则

### 1. 表信息验证
- **表名称**：必填，长度不超过50个字符
- **表类型**：默认为“多维表”，不可修改
- **表描述**：可选，长度不超过200个字符

### 2. 字段信息验证
- **字段名称**：必填，长度不超过30个字符
- **字段类型**：必填，必须是系统支持的类型
- **字段属性**：根据字段类型的不同，有不同的验证规则
  - **文本**：可设置默认值，可选
  - **数字**：可设置默认值，可选
  - **单选**：必须设置选项列表，至少1个选项
  - **多选**：必须设置选项列表，至少1个选项
  - **超级链接**：可设置默认值，可选
  - **日期/时间**：可设置默认值，可选
  - **布尔值**：可设置默认值（true/false），可选
