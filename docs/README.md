# 基于SQLite的多维表项目

## 项目概述

本项目是一个基于SQLite文件的多维表应用，使用Rust作为核心解析引擎，提供统一的接口，同时支持Flutter（移动端）和Electron（桌面端）作为应用展示层。

## 技术栈

- **核心引擎**：Rust（用于SQLite解析和提供接口）
- **移动端**：Flutter
- **桌面端**：Electron
- **数据存储**：SQLite

## 核心功能

- **多维表数据管理**：支持创建、编辑、删除多维表结构
- **SQLite数据存储**：使用SQLite文件作为存储介质，确保数据安全和一致性
- **跨平台支持**：同时支持移动端和桌面端应用
- **统一接口**：Rust提供统一的数据操作接口，确保各平台功能一致性

## 项目目标
