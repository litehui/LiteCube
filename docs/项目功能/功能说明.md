# 项目功能说明

## 核心功能

### 1. 多维表数据管理
- **表结构管理**：创建、编辑、删除多维表结构
- **字段管理**：支持多种字段类型（文本、数字、日期、布尔值等）
- **数据操作**：添加、编辑、删除表数据
- **数据查询**：支持复杂条件查询和筛选

### 2. SQLite数据存储
- **文件管理**：创建、打开、保存SQLite文件
- **数据持久化**：确保数据安全存储和一致性
- **事务支持**：支持数据操作的事务管理
- **备份与恢复**：支持数据备份和恢复功能

### 3. 跨平台支持
- **移动端**：Flutter实现的移动应用
- **桌面端**：Electron实现的桌面应用
- **数据同步**：支持不同平台间的数据同步

### 4. 统一接口
- **Rust核心**：使用Rust实现SQLite解析和数据操作
- **API设计**：提供统一的数据操作接口
- **跨平台兼容**：确保各平台功能一致性

## 高级功能

### 1. 数据可视化
- **表格视图**：传统表格形式展示数据
- **卡片视图**：卡片形式展示数据
- **图表分析**：数据统计和图表展示

### 2. 协作功能
- **数据共享**：支持数据文件共享
- **版本控制**：记录数据变更历史

### 3. 自动化
- **公式计算**：支持字段间的公式计算
- **数据验证**：支持数据输入验证
- **批量操作**：支持批量数据操作

### 4. 安全管理
- **数据加密**：支持SQLite文件加密
- **访问控制**：支持基本的访问权限控制
