# 项目功能说明

## 核心功能

### 1. 多维表数据管理
- **表结构管理**：创建、编辑、删除多维表结构
- **字段管理**：支持多种字段类型，包括：
  - 文本：支持长文本输入
  - 数字：自动兼容整数与浮点数，保证浮点数相关计算准确性
  - 单选：预设选项列表，用户选择其中一个
  - 多选：预设选项列表，用户可选择多个
  - 超级链接：支持输入和显示超链接
  - 日期/时间：支持日期和时间输入
  - 布尔值：支持 true/false 选择
- **数据操作**：添加、编辑、删除表数据
- **数据查询**：支持复杂条件查询和筛选
- **编辑体验**：实现像WPS Excel一样丝滑的表格编辑体验
- **功能丰富度**：提供像飞书多维表格一样丰富的功能

### 2. SQLite数据存储
- **文件管理**：创建、打开、保存SQLite文件
- **数据持久化**：确保数据安全存储和一致性
- **事务支持**：支持数据操作的事务管理
- **备份与恢复**：支持数据备份和恢复功能

### 3. 跨平台支持
- **移动端**：Flutter实现的移动应用
- **桌面端**：Electron实现的桌面应用
- **命令行**：Rust CLI实现的命令行工具
- **数据同步**：支持不同平台间的数据同步

### 4. 统一接口
- **Rust核心**：使用Rust实现SQLite解析和数据操作
- **API设计**：提供统一的数据操作接口
- **跨平台兼容**：确保各平台功能一致性

## 高级功能

### 1. 数据可视化
- **表格视图**：传统二维表形式展示数据
- **甘特图**：时间线和任务进度展示
- **看板视图**：任务和状态的看板展示
- **画册视图**：图片和媒体内容的画册展示
- **图表分析**：数据统计和图表展示，包括饼状图、条形图、柱状图等多种图表类型

### 2. 协作功能
- **数据共享**：支持数据文件共享
- **版本控制**：记录数据变更历史

### 3. 自动化
- **公式计算**：支持字段间的公式计算
- **数据验证**：支持数据输入验证
- **批量操作**：支持批量数据操作

### 4. 安全管理
- **数据加密**：支持SQLite文件加密
- **访问控制**：支持基本的访问权限控制
